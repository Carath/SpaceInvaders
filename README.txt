***********************************************************
INSTALLATION: (Linux)
***********************************************************
Taper les commandes suivantes dans la console:

sudo apt-get install libsdl2
sudo apt-get install libsdl2-image
sudo apt-get install libsdl2-ttf
sudo apt-get install libfreetype6

Puis se placer dans l'arborescence du dossier du jeu (e.g cd Document/space_project) et taper:

make clean
make

***********************************************************
LANCEMENT:
***********************************************************
Une fois que l'installation a été effectuée, se placer dans l'arborescence du dossier du jeu, et taper:

./space_invaders

***********************************************************
LIMITATIONS:
***********************************************************
Unique résolution supportée: 1000 x 1000
Caractères ASCII seulement.
Contrôles hardcodés.
